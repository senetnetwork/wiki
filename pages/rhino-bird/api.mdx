## RB API
**API Endpoint**
```
https://rb-api.senetnetwork.io
```
## Bind Senet Account

> POST /api/identityBind

> Body Parameters

```json
{
  "username": "string",
  "password": "string",
  "uuid": "string"
}
```

### Request Parameters

| Name      | NA     | NA     | Mandatory | Note                |
| --------- | ------ | ------ | --------- | ------------------- |
| sign      | header | string | Yes       | signature           |
| timestamp | header | string | Yes       | timestamp           |
| appId     | header | string | Yes       | appId               |
| body      | body   | object | YES       | none                |
| username  | body   | string | Yes       | senet username      |
| password  | body   | string | Yes       | senet auth token    |
| uuid      | body   | string | Yes       | third-party user id |

> Example

> Success

```json
{
  "success": true,
  "message": "success",
  "code": 200,
  "result": {
    "openId": "WMQbgrdXpxGwqabvdsndsndmWdijTVJ"
  },
  "timestamp": 1681723477191
}
```

### Return Data Structure

Status Code **200**

| Name      | Type    | Mandatory | Note        |
| --------- | ------- | --------- | ----------- |
| success   | boolean | true      | none        |
| message   | string  | true      | none        |
| code      | integer | true      | none        |
| result    | object  | true      | none        |
| openId    | string  | true      | user openId |
| timestamp | integer | true      | none        |
